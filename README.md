# 🗺️ 경기도 지역화폐 가맹점 지도 (Gyeonggi Local Currency Map)

이 프로젝트는 경기도 공공데이터 포털의 API를 활용하여 경기도 내 지역화폐 가맹점 정보를 지도 위에 시각화하고, 사용자 편의를 위한 다양한 필터링 기능을 제공하는 웹 애플리케이션입니다.

---

## 🚀 개발 프로세스 요약 (작성 순서)

이 프로젝트는 다음과 같은 단계별 과정을 거쳐 정교하게 구축되었습니다:

### 1. 환경 설정 및 기반 구축 (Setup)
*   **Vite + React + TypeScript**: 최신 개발 환경을 구축하고 타입 안정성을 확보했습니다.
*   **라이브러리 구성**: `axios`(통신), `@tanstack/react-query`(데이터 캐싱), `react-kakao-maps-sdk`(지도), `tailwindcss v4`(스타일), `lucide-react`(아이콘)를 설치하고 설정했습니다.
*   **보안 설정**: `.env` 파일을 통해 API 키를 분리 관리하고, `vite.config.ts`에서 개발 포트를 `1205`로 커스터마이징했습니다.

### 2. 데이터 페칭 레이어 구현 (Data Layer)
*   **API 서비스**: `src/api/gyeonggiApi.ts` 파일을 생성하여 공공데이터 포털과 통신하는 로직을 작성했습니다.
*   **타입 정의**: 샘플 XML 데이터를 분석하여 `MerchantRow` 등 정교한 TypeScript 인터페이스를 정의했습니다.
*   **커스텀 훅**: `useMerchants` 훅을 만들어 데이터 로딩 상태와 에러를 효율적으로 관리하게 했습니다.

### 3. 지도 및 마커 시각화 (Map Integration)
*   **카카오맵 연동**: `index.html`에 SDK를 로드하고 `Map.tsx`에서 지도를 렌더링했습니다.
*   **마커 클러스터링**: 1,000개가 넘는 방대한 마커를 효율적으로 보여주기 위해 `clusterer` 라이브러리를 적용했습니다.
*   **인포윈도우(Overlay)**: 가맹점을 클릭했을 때 이름, 상세주소, 업종, 전화번호, 그리고 **사업자 상태(정상/휴업)**를 보여주는 커스텀 오버레이를 디자인했습니다.

### 4. 고급 필터링 기능 추가 (Advanced Features)
*   **가객점 검색**: 이름이나 업종 키워드로 실시간 필터링하는 기능을 추가했습니다.
*   **분위기 있는 디자인**: Tailwind CSS v4를 사용하여 세련된 대시보드 형태의 UI를 완성했습니다.
*   **'전체표시' 고도화**: 경기도 전역의 데이터를 고르게 보여주기 위해 31개 시/군에 대해 병렬 API 요청을 수행하는 분산 페칭 로직을 구현했습니다.
*   **자동 센터링**: 도시 선택이나 '전체표시' 모드에 따라 지도의 중심점과 줌 레벨이 자동으로 조절되도록 최적화했습니다.

---

## ✨ 주요 기능 (Key Features)

*   **실시간 데이터 연동**: 경기도 공공데이터 포털의 최신 가맹점 정보 반영.
*   **분산형 전체 보기**: 경기도 전역의 가맹점을 뭉침 없이 고르게 시각화.
*   **스마트 검색**: 가맹점명 및 업종 기반의 빠른 검색.
*   **클러스터링 UI**: 지도 확대/축소에 따라 마커들을 직관적으로 그룹화.
*   **상태 정보 제공**: 계속사업자(초록색)와 휴/폐업자를 구분하여 안전한 거래 정보 제공.

## 🛠️ 기술 스택 (Tech Stack)

*   **Frontend**: React 19, TypeScript, Vite
*   **State Management**: TanStack Query (React Query) v5
*   **Styling**: Tailwind CSS v4
*   **Map API**: Kakao Maps JavaScript API
*   **Icons**: Lucide React

## ⚠️ 실행 시 주의사항

이 프로젝트를 로컬에서 실행하기 위해서는 `.env` 파일에 다음 두 가지 키가 필요합니다 (보안상 외부 발설 금지):

1.  `VITE_GG_API_KEY`: 경기도 공공데이터 포털에서 발급받은 인증키.
2.  `VITE_KAKAO_MAP_API_KEY`: 카카오 개발자 센터에서 발급받은 **JavaScript 키**.

또한, 카카오 개발자 센터의 플랫폼 설정에서 `http://localhost:1205` 도메인이 등록되어 있어야 지도가 정상적으로 표시됩니다.
